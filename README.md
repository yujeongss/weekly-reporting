# 전략개발팀 주간업무보고 반자동화 도구

전략개발팀의 주간업무보고 작성 작업을 반자동화하는 단일 HTML 파일 기반 웹 도구입니다.

---

## 프로젝트 소개

매주 반복되는 주간업무보고 작성을 효율화하기 위해 만들어진 도구입니다. 팀원 각자가 브라우저에서 과제, 운영/요청, 제품 배포 데이터를 입력하면 기존 양식에 맞는 Excel 파일을 자동으로 생성합니다. 서버 없이 `index.html` 파일 하나만으로 동작합니다.

---

## 핵심 기능

- **팀원별 탭 관리**: 팀원별로 분리된 탭에서 독립적으로 데이터 입력
- **과제 관리**: 진행 중인 과제의 금주 실적 및 차주 계획 CRUD
- **운영/요청 관리**: 운영 및 요청 업무 항목 CRUD
- **제품 배포 관리**: 배포 일정 및 내용 CRUD
- **Excel 자동 생성**: 입력한 데이터를 기존 보고 양식 템플릿에 채워 `.xlsx` 파일로 내보내기
- **로컬 자동 저장**: 페이지를 닫아도 데이터가 브라우저에 보존됨

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 아키텍처 | Single HTML file (빌드 불필요) |
| 언어 | Vanilla JavaScript (프레임워크 없음) |
| Excel 처리 | [ExcelJS 4.4.0](https://github.com/exceljs/exceljs) (CDN) |
| 데이터 저장 | localStorage (`weekly_report_data_v1`) |
| 배포 방식 | 파일 직접 실행 (`file://` 프로토콜) |

---

## 사용법

1. `index.html`을 브라우저에서 열기 (더블클릭 또는 드래그&드롭)
2. 상단 탭에서 본인 이름 탭 선택
3. 과제 / 운영·요청 / 제품 배포 섹션에 데이터 입력
4. 우측 상단 **Excel 내보내기** 버튼 클릭
5. 생성된 `.xlsx` 파일 확인 및 제출

> 데이터는 자동으로 브라우저 localStorage에 저장되므로 별도 저장 버튼이 필요 없습니다.

## Phase 로드맵

### Phase 1 — 개인 탭 + 일감관리 + Excel 내보내기 (완료)

팀원별 탭 UI 구성, 과제/운영/배포 CRUD, 템플릿 기반 Excel export 구현.

### Phase 2 — 팀 시트 자동 취합 (예정)

전체 팀원 데이터를 하나의 팀 시트로 자동 취합. 금주 실적 / 차주 계획 / 프로젝트 진척도 / 주요이슈 섹션 구성.

### Phase 3 — AI 챗봇 패널 (예정)

우측 패널 AI 채팅 UI 추가. Claude API 연동으로 보고서 자동 요약 및 작성 지원.

## 데이터 저장

- **저장 위치**: 브라우저 localStorage
- **키 이름**: `weekly_report_data_v1`
- **예상 용량**: 6명 팀 기준 약 50KB (localStorage 5MB 한도 내)
- **주의**: 브라우저 캐시/데이터 삭제 시 초기화됨. 중요한 데이터는 Excel로 내보내기 후 보관 권장.
